## Knowledge Base Engines

> RAG frameworks, knowledge base platforms, and AI-powered search engines.

- [PieKBS](https://pieteams.github.io/piekbs/) - Knowledge base engine for AI agents with advanced retrieval capabilities.
- [RAGFlow](https://github.com/infiniflow/ragflow) - An open-source RAG engine based on deep document understanding.
- [Dify](https://github.com/langgenius/dify) - Open-source LLM app development platform with knowledge base features.
- [FastGPT](https://github.com/labring/FastGPT) - Open-source AI knowledge base platform built on LLMs.
- [MaxKB](https://github.com/1Panel-dev/MaxKB) - Ready-to-use and flexible knowledge base based on LLMs and RAG.
- [AnythingLLM](https://github.com/Mintplex-Labs/anything-llm) - All-in-one desktop & Docker AI application with built-in RAG.
- [Khoj](https://github.com/khoj-ai/khoj) - AI research assistant and knowledge base for your notes and documents.
- [PrivateGPT](https://github.com/zylon-ai/private-gpt) - Interact with your documents using the power of LLMs.
- [LocalGPT](https://github.com/punkpeye/localgpt) - Chat with your documents using LLMs with 100% privacy.
- [Verba](https://github.com/weaviate/verba) - Retrieval-Augmented Generation (RAG) chatbot powered by Weaviate.
- [Quivr](https://github.com/QuivrHQ/quivr) - Your second brain, powered by AI with RAG.
- [DB-GPT](https://github.com/eosphoros-ai/DB-GPT) - AI-powered data app development framework with knowledge base support.
- [OpenWebUI](https://github.com/open-webui/open-webui) - ChatGPT/Ollama web UI with knowledge base features.
- [Ragas](https://github.com/explodinggradients/ragas) - Evaluation framework for RAG pipelines.

## Knowledge Graph Tools

> Tools for building, managing, and querying knowledge graphs with AI.

- [GraphRAG](https://github.com/microsoft/graphrag) - Microsoft's graph-based RAG framework for enhanced retrieval.
- [Nano-GraphRAG](https://github.com/HKUDS/nano-graphrag) - Lightweight and fast GraphRAG implementation.
- [GraphRAG-API](https://github.com/neo4j/neo4j-graphrag-python) - Python library for Graph RAG with Neo4j.
- [LangChain GraphRAG](https://github.com/langchain-ai/langchain) - Graph RAG components in LangChain framework.
- [LightRAG](https://github.com/HKUDS/LightRAG) - Simple and fast retrieval-augmented generation with knowledge graphs.
- [MindGraph](https://github.com/cpacker/MindGraph) - Knowledge graph construction and reasoning for LLMs.
- [Graphene](https://github.com/graphene-app/graphene) - Open-source graph database for AI knowledge graphs.
- [Neo4j](https://github.com/neo4j/neo4j) - The world's leading graph database for knowledge graphs.
- [Apache AGE](https://github.com/apache/age) - PostgreSQL extension for graph databases.
- [TigerGraph](https://github.com/tigergraph/tigergraph) - High-performance graph database for knowledge graph applications.

## Vector Databases

> Databases optimized for storing and querying vector embeddings.

- [Chroma](https://github.com/chroma-core/chroma) - AI-native open-source embedding database.
- [Weaviate](https://github.com/weaviate/weaviate) - Vector database for semantic search and AI applications.
- [Milvus](https://github.com/milvus-io/milvus) - Open-source vector database for AI applications.
- [Pinecone](https://github.com/pinecone-io/pinecone-python-client) - Vector database for AI-native applications.
- [Qdrant](https://github.com/qdrant/qdrant) - Open-source vector similarity search engine with extended filtering support.
- [FAISS](https://github.com/facebookresearch/faiss) - Facebook AI Similarity Search library.
- [Annoy](https://github.com/spotify/annoy) - Approximate Nearest Neighbors in C++/Python.
- [Vald](https://github.com/vdaas/vald) - Cloud-native distributed vector search engine.
- [Vespa](https://github.com/vespa-engine/vespa) - Big data serving engine with vector search capabilities.
- [Marqo](https://github.com/marqo-ai/marqo) - Open-source vector search engine for AI applications.
- [LanceDB](https://github.com/lancedb/lancedb) - Open-source vector database built on Lance format.
- [Pgvector](https://github.com/pgvector/pgvector) - Open-source vector similarity search for Postgres.

- [Filesystem MCP](https://github.com/modelcontextprotocol/servers/tree/main/src/filesystem) - MCP server for file-based knowledge access.
- [PostgreSQL MCP](https://github.com/modelcontextprotocol/servers/tree/main/src/postgres) - MCP server for database knowledge queries.
- [Memory MCP](https://github.com/modelcontextprotocol/servers/tree/main/src/memory) - MCP server for persistent memory and knowledge storage.
- [Notion MCP](https://github.com/modelcontextprotocol/servers/tree/main/src/notion) - MCP server for Notion knowledge base integration.
- [GitHub MCP](https://github.com/modelcontextprotocol/servers/tree/main/src/github) - MCP server for GitHub knowledge and code access.
- [Slack MCP](https://github.com/modelcontextprotocol/servers/tree/main/src/slack) - MCP server for Slack knowledge retrieval.

## LLM Wiki Tools

> Tools for building and managing wiki-style knowledge bases for LLMs.

- [WikiLLM](https://github.com/wiki-llm/wikillm) - Framework for building LLM-powered wiki systems.
- [GPTWiki](https://github.com/gptwiki/gptwiki) - AI-powered wiki platform with LLM integration.
- [DocuChat](https://github.com/docuchat/docuchat) - Chat with your documentation using LLMs.
- [MkDocs](https://github.com/mkdocs/mkdocs) - Static site generator for project documentation.
- [Docusaurus](https://github.com/facebook/docusaurus) - Easy to maintain open source documentation websites.
- [Read the Docs](https://github.com/readthedocs/readthedocs.org) - Automated documentation building and hosting.
- [Outline](https://github.com/outline/outline) - Team knowledge base and wiki with AI features.
- [BookStack](https://github.com/BookStackApp/BookStack) - A platform to store and organize information.
- [Wiki.js](https://github.com/requarks/wiki) - Modern and powerful wiki app built with Node.js.
- [Dendron](https://github.com/dendronhq/dendron) - Personal knowledge management system for VS Code.

## Document Processing & Indexing

> Tools for parsing, chunking, and indexing documents for knowledge bases.

- [LangChain Document Loaders](https://github.com/langchain-ai/langchain/tree/master/libs/community/langchain_community/document_loaders) - Comprehensive document loading utilities.
- [LlamaIndex](https://github.com/run-llama/llama_index) - Data framework for LLM applications with document indexing.
- [Unstructured](https://github.com/Unstructured-IO/unstructured) - Open source pre-processing and extraction tools for unstructured data.
- [PyMuPDF](https://github.com/pymupdf/PyMuPDF) - High-performance Python PDF toolkit.
- [Apache Tika](https://github.com/apache/tika) - Content detection and extraction toolkit.
- [Marker](https://github.com/VikParuchuri/marker) - Fast, high quality PDF to markdown conversion.
- [Docling](https://github.com/DS4SD/docling) - AI-powered document understanding and conversion.
- [PyPDF](https://github.com/py-pdf/pypdf) - A pure-python PDF library capable of splitting, merging, and transforming.
- [pdfplumber](https://github.com/jsvine/pdfplumber) - Plumb a PDF for detailed extraction of text and table data.
- [MegaParse](https://github.com/QuivrHQ/MegaParse) - Efficient and flexible parsing tool for LLMs.
- [Chonkie](https://github.com/6gh/Chonkie) - Fast, lightweight, and efficient text chunking library.
- [LlamaCpp](https://github.com/ggerganov/llama.cpp) - LLM inference in C/C++ with document processing capabilities.
